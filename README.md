# Rentiva Backend

Express + Mongoose API για τη Rentiva.

## Απαιτήσεις
- Node.js 18+
- MongoDB Atlas URI

## Γρήγορη εκκίνηση
```bash
# 1) Εγκατάσταση
npm install

# 2) Δημιούργησε .env από το .env.example και συμπλήρωσε MONGO_URI/PORT
cp .env.example .env
# (άνοιξε .env και βάλε τις τιμές σου)

# 3) Τρέξε σε dev
npm run dev
